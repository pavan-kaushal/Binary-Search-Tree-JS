# Binary-Search-Tree-JS
implementation of BST using JavaScript and HTML

# Features
### 1 . every element is uniquely identified by a hash
 * hash of root = ""
 * hash of left child = hash of parent + "0"
 * hash of right child = hash of parent + "1"
